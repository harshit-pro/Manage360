# Manage360
Multi-tenant Study Library Management SaaS platform

A multi-tenant SaaS platform for study library owners to manage
students, seats, memberships, fees, and analytics.

## Tech Stack
- Java 17, Spring Boot 3
- PostgreSQL
- JWT Security
- React Admin Portal
- Docker

## Architecture
Modular Monolith (designed to evolve into microservices)
